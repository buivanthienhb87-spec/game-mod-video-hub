# FAQ

## What types of videos belong in this catalog?

Tutorials, installation walkthroughs, before-and-after showcases, configuration guides, and creator introductions. All entries should link to legitimate sources.

## Can I add my own mod videos?

Yes. Follow the template in `templates/video-entry.md` and open a pull request.

## Does this project host video files?

No. This project stores metadata and links. Video files are hosted by creators on their chosen platforms.
