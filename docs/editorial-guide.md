# Editorial Guide

Video entries should help viewers understand what a tutorial demonstrates, what version it applies to, and where the source material can be reviewed. The catalog should not host unsafe files or amplify misleading claims.

| Field | Standard |
|---|---|
| Title | Clear and descriptive. |
| Creator | Respect the original creator attribution. |
| Source URL | Link to an official or creator-controlled page. |
| Category | Use consistent lowercase categories. |
| Reviewed date | Record the latest verification date. |
