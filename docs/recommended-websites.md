# Recommended Modding, Video, and Creator Resource Websites

This page lists reputable resources for documenting mod tutorial videos, creator attribution, and safe installation guidance. Maintainers should prefer official or creator-controlled links when adding catalog entries.

| Website | Recommended Use |
|---|---|
| [Nexus Mods](https://www.nexusmods.com/) | Major mod hosting platform with project pages, creator metadata, and update histories. |
| [Mod DB](https://www.moddb.com/) | Long-running community platform for mods, previews, and development diaries. |
| [Steam Workshop](https://steamcommunity.com/workshop/) | Official Steam community platform for supported game mods and workshop items. |
| [YouTube Creator Academy](https://www.youtube.com/creators/) | Official creator resource for video publishing and channel management guidance. |
| [Creative Commons](https://creativecommons.org/) | License reference for media attribution and reuse rules. |
| [OBS Studio](https://obsproject.com/) | Open-source recording and streaming software for creating tutorial videos. |
| [FFmpeg Documentation](https://ffmpeg.org/documentation.html) | Official documentation for media processing workflows. |
| [GitHub Docs](https://docs.github.com/) | Official GitHub documentation for repository management and contribution workflows. |

## Review Policy

Recommended websites should be reviewed periodically. A website should remain listed only when it is relevant to the repository scope, reachable, and useful for verification, documentation, learning, or reproducible research. Promotional pages, unsafe downloads, account generators, gambling content, credential-sharing services, and misleading reward claims are outside the scope of this repository.
